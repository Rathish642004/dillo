# Section

A full-bleed band with the standard 1240px inner container and consistent
vertical rhythm (96px on `lg`, 64px on `md`). Use `tone="navy"` for dark
bands (stats, footer-adjacent CTAs) and `bottomRule` to drop the signature
red rule at the bottom edge.

```jsx
<Section tone="alt" bottomRule>
  <SectionHeader eyebrow="Our products" title="Five verticals, one quality bar." />
</Section>
```
