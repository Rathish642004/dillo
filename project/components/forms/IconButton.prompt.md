# IconButton

A square button that holds a single icon and nothing else. Always pass an
`aria-label`. Use **navy** for the floating WhatsApp / call buttons that
appear on dark sections, **secondary** for nav toggles, **ghost** for
toolbar density.

```jsx
<IconButton aria-label="Call us" variant="navy" size="lg">
  <Phone size={20} />
</IconButton>
```
