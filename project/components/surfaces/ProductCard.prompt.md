# ProductCard

The category card used on the Home "Our Products" grid and in the Products
listing. Image on top with a red "UNIFORMS" tag overlay; title + description
+ optional item chips; ghost CTA at the bottom; signature red bar appears on
hover.

```jsx
<ProductCard
  title="Healthcare uniforms"
  description="Scrubs, lab coats, ward shirts — built for a hundred wash cycles."
  imageSrc="/img/healthcare.jpg"
  items={["Scrubs", "Lab coats", "Ward shirts"]}
  ctaLabel="Explore"
/>
```
