# TestimonialCard

Serif italic quote card with an oversized red opening glyph in the corner
and a small avatar + role line at the bottom. Used in the "Client
Testimonials" home band.

```jsx
<TestimonialCard
  quote="Six hospitals, same fabric grade across all of them. Dillo just gets it."
  author="Dr. Anita Krishnan"
  role="Procurement Head, MedStar Hospitals"
/>
```
