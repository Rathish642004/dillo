# Marketing Website UI kit

An interactive recreation of the full Dillo Uniforms public site.

**Pages**
- `HomePage.jsx` — Hero · Trusted By · Why Dillo · Numbers · Products · How It Works · Testimonials · CTA
- `AboutPage.jsx` — Story · Experience · Industries · Manufacturing · CTA
- `ProductsPage.jsx` — Sub-nav across 5 categories, hero, item grid, specs, CTA
- `PortfolioPage.jsx` — Filterable grid of case studies (client name, location, quantity, year)
- `ContactPage.jsx` — Quote form, contact details, WhatsApp deep link, map embed

Open `index.html` to navigate between them. The state is local React — no
routing library, no backend.

**Images** are loaded from Unsplash CDN (see `photos.js`). Swap to client
photography by editing that single map.
